## Run Locally

Clone the project

```bash
  git clone https://github.com/hamzasaleem2/chatgpt95.git
```
Install dependencies

```bash
  bun install
```

Start the server

```bash
  bun run dev
```


## .env

create `.env` file in project dir. and add:

`OPENAI_API_KEY=sk-xxxxx`
